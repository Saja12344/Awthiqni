موقع بورتفوليو شخصي لـ **نوره ماسoud العقيلي** — متخصصة في Product Discovery و User Research.

Personal portfolio website for **Norah Masoud Aloqayli**, Product Discovery & User Research specialist.

## Live Site

**https://saja12344.github.io/portfolio-norah/**

(بعد تفعيل GitHub Pages — راجع `ENABLE_PAGES.md`)

## Features

- LinkedIn-inspired black header with custom Noon logo
- Purple and grey theme with black accents
- Animated typewriter role display
- Experience, skills, and featured projects sections
- Official Behance icon in contact section
- Responsive design for mobile and desktop

## Projects Showcased

- **Siraj** — Emergency rescue (1st Place, The Garage Challenge)
- **GrowTube** — Customer discovery for content creators
- **Nasagh** — Hydration habits during Ramadan
- **Abyan** — App redesign through user research

## Run Locally

Open `index.html` in a browser, or serve with any static server:

```bash
npx serve .
```
